# CSI606-2025-01 - Remoto - Proposta de Trabalho Final

**Discente:** João Pedro Ferreira Lobo

## Resumo
O **Ploton** é uma aplicação web voltada para a organização financeira pessoal. O objetivo central é consolidar, em uma interface moderna e de alta legibilidade, o balanço entre rendas, despesas e patrimônio investido. A plataforma permitirá que o usuário acompanhe sua saúde financeira em tempo real através de um painel de controle centralizado que resume transações e metas.

## 1. Tema
O trabalho final tem como tema o desenvolvimento de um sistema full-stack de gestão financeira, utilizando React no frontend, Spring Boot no backend e PostgreSQL como banco de dados.

## 2. Escopo
Este projeto terá as seguintes funcionalidades:
* **Painel Principal (Dashboard):** Visualização resumida de saldo (renda, gastos e sobras), gráfico de faturas e resumo de investimentos.
* **Módulo de Transações:** Registro e listagem de entradas e saídas.
* **Módulo de Orçamentos:** Definição de limites de gastos por categoria.
* **Módulo de Metas:** Acompanhamento de objetivos de economia.
* **Módulo de Investimentos:** Monitoramento simplificado de ativos.
* **Módulo Livelo:** Espaço para gestão manual de pontuação em programas de fidelidade.

## 3. Restrições
Neste trabalho não serão considerados:
* Integração automatizada com contas bancárias via Open Finance (os dados serão manuais).
